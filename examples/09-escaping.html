Prose Escapes

A literal hash: # not a macro.

A literal bracket: [ not a call.

A literal closing bracket: ] in text.

A literal colon: : not a body separator.

A literal equals: = not an argument.

A literal backslash: \ stands alone.

Unicode Escapes in Prose

Copyright symbol: ©

Em dash: —

String Escapes

A double quote: " inside a string.

A newline: in a string.

A tab: in a string.

A backslash: \ in a string.

Hex Escape in String

Copyright: © symbol.

Raw String Contrast

Escapes are literal in raw strings: \n \t \\ \# \[
